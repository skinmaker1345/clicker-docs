# Clicker
투표, 공지, 그리고 버튼을 통한 자동 역할 추가를 할 수 있는 봇입니다.

## 🛠️ 기능
- 투표 기능
- 임베드, 메시지를 사용한 공지 기능
- 버튼으로 자동 역할 추가 / 삭제 기능
